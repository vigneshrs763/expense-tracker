# 💰 Expense Tracker

A web-based Expense Tracker application that helps users manage their income and expenses efficiently. Users can track transactions, monitor spending habits, and maintain a clear overview of their financial activities.

## 🚀 Features

- Add income and expense transactions
- View transaction history
- Calculate total balance
- Track total income
- Track total expenses
- Simple and responsive user interface

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript


## 🎯 Key Functionalities

- Real-time balance calculation
- Transaction management
- Expense tracking
- Income tracking
- User-friendly interface
